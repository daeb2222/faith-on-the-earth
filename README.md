### Option B â€” Manual Setup

**Prerequisites:**

* Python 3.12+
* `uv` package manager
* API keys for Mistral AI and Rodin AI (optional)

**1. Configure environment**

```bash
cd Backend
cp .env.template .env

```

Fill in your keys in the `.env` file:

```ini
MISTRAL_API_KEY=...
RODIN_API_KEY=...

```

**2. Install dependencies**

```bash
uv sync

```

**3. Run the server**

```bash
uv run python main.py

```
