# Installation

1. Unzip the .zip file to any folder you like.

2. Open Unity 2023.x or later and go to "Window" -> "Package Manager".

3. Click on the "+" button on the top left corner and select "Install package from disk...", then select the "package.json" file in the unzipped folder.

4. Wait for the package to be imported and click on the "Rodin" Tab in the "Tools" section, and you should see the plugin window.

# Dependencies

Unity 2023.x or later, and the following packages:

- Newtonsoft Json
- com.unity.importer.usd
